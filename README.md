# CLUBE DO TERMUX

Painel multifunções compatível com **Termux (Android)** e **Linux (Debian/Ubuntu/Fedora/Arch)**.
37 opções automatizadas: instalações, rede, segurança, IA, servidores e utilitários.

## 📦 Instalação

```bash
# Instalação direta (Termux ou Linux)
bash -c "$(curl -fsSL https://raw.githubusercontent.com/carlos46743/clube-do-termux/master/termux-panel.sh)"
```

```bash
# Ou baixe e execute
curl -fsSL https://raw.githubusercontent.com/carlos46743/clube-do-termux/master/termux-panel.sh -o termux-panel.sh
bash termux-panel.sh
```

## 🔧 Requisitos

**Termux:**
- Termux instalado no Android
- Conexão com internet
- 2GB+ de espaço livre

**Linux:**
- Debian/Ubuntu, Fedora, Arch ou openSUSE
- `curl` instalado
- `sudo` configurado
